项目简介 Project Introduction

此项目初衷为帮助我自己写小说的时候确立时间线（青陵元历），因为似乎没有什么好用的可以自己定制时间线时间类型的软件。

The original intention of this project is to assit me to list the timeline when I am writing a novel, as there are few timeline softwares that allow me to create the type of the time by myself.

名为“贞元”是因为该词可以借指时令的周而复始和天道人事的转换，既有时间变迁的意味，同时又带有着我个人比较偏爱的轮回的史诗感。

The name "TempusLinea" is the combination of the latin words "Tempus" and "Linea", which means "time" and "line" in English, as you have already guessed.

项目主要使用electron.js搭建而成。本人学艺不精，尚是高中生，学习过程难谈规范，多以试验为主，所以冗余低效代码颇多，还望各位大佬海纳。

This project mainly use electron.js. I have to say that my program skill is not so good. As a high-school student, my learning experience was no where near "professional". In fact, I mostly learned it by testing again and again. Hence, this project is full of inefficient codes, which I hope you can accept.

还需要感谢一下开源字体LxgwBright的开发者，为本项目设计惨烈的页面勉强提了提颜值；和开源标志库feather，本项目中出现的所有按钮的标志全部是从那里下载的。

In addition, I would like to show my gratitude to the developers of the open-source font LxgwBright, as the font improved the appereance of this project significantly. And I would also express my gratitude to the feather icon repo, where all of the icons of the buttons in this project are downloaded from.

在最后，该项目已经完全开源，各位可以自行从Github下载源码或者从我的个人网站（尚未搭建完成）下载打包好的软件。请随意基于这个软件进行的非盈利、非商业的开发，如果你在这么做之前告知我一下，我会非常感谢，虽然这不是必须的。而若是商业开发请通过邮件联系我并经得我的同意。
